# Around the U.S. (Portuguese Version)

This is a responsive web application developed as part of the TripleTen Software Engineering bootcamp. The project showcases a social networking style landing page where users can share photos of places they've visited, like posts, and edit their profiles.

## 🚀 Features

- **Profile Editing**: Users can update their name and professional description.
- **Card Management**: Ability to add new cards with titles and image links.
- **Interactivity**: 
  - Like button functionality.
  - Delete button to remove cards.
  - Image popup for full-size viewing.
- **Responsive Design**: The layout is fully responsive, optimized for desktop, tablet, and mobile devices (using BEM methodology).
- **Form Validation**: Client-side validation to ensure data integrity before submission.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Layout and styling using:
  - Flexbox and Grid.
  - BEM (Block Element Modifier) methodology.
  - Media Queries for responsiveness.
- **JavaScript (Vanilla)**: 
  - DOM manipulation.
  - Event listeners.
  - Modal logic (opening/closing/closing on Overlay or Esc).
- **Figma**: The project was built based on a professional design brief.

## 📂 Project Structure

```text
.
├── blocks/          # CSS files organized by BEM blocks
├── images/          # Image assets and icons
├── pages/           # Main CSS file (index.css)
├── scripts/         # JavaScript logic (index.js, validate.js, etc.)
├── vendor/          # Third-party fonts and reset files
└── index.html       # Main entry point
```
